Detailed Description

Message Wrapper structure.

The actual message that is being send and contains the message type and the message structure. Type can be one of:

  • 1: Intro message
  • 2: Welcome message
  • 3: Request Config message
  • 4: Config message
  • 5: Data message
  • 6: Error message
