85// Examples of messages:
86//
87// ###S1/my-topic/status : Subscibe to topic 'my-topic/status' with QOS 1
88// ###U/my-topic/status : Unsubscibe from topic 'my-topic/status'
89// ###P10/my-topic/status#P#data : Publish payload 'data' to topic 'my-topic/status' with QOS 1
90// and retained flag set to false
91// ###Rmy-topic/status#P#data : A message with payload 'data' on topic 'my-topic/status'
92// was received from the MQTT broker en is send back to the
93// controller.
94// ###B : Reboot the target client device
95//
96// Notes:
